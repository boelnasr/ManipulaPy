Notebook Course: Robotics with ManipulaPy
==========================================

A foundations-first, executable course that teaches robotics *through* ManipulaPy on the
Franka Panda. Each notebook pairs the concept (with light derivation) with a live
demonstration using the library, and every figure is rendered from TikZ source.

Each notebook carries an **Open in Colab** badge so you can run it in the cloud with no
local setup. The pages below are the executed notebooks with their committed outputs.
